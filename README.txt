HOW TO RUN IN VISUAL STUDIO CODE:
---------------------------------
1. CREATE FOLDER:
   - Create a new folder on your computer
   - Name it "alex-all-trade-website"

2. SAVE ALL FILES:
   - Save index.html, style.css, script.js, and README.txt
   - All files should be in the SAME FOLDER

3. OPEN IN VS CODE:
   - Open Visual Studio Code
   - Click "File" → "Open Folder"
   - Select the "alex-all-trade-website" folder

4. INSTALL LIVE SERVER:
   - In VS Code, click Extensions icon (Ctrl+Shift+X)
   - Search for "Live Server" by Ritwick Dey
   - Click "Install"

5. RUN THE WEBSITE:
   - In VS Code, right-click on "index.html"
   - Select "Open with Live Server"
   - Website opens at: http://localhost:5500

ALTERNATIVE METHODS:
--------------------
1. DIRECT BROWSER OPEN:
   - Double-click "index.html" file
   - Opens in your default browser

2. VS CODE PREVIEW:
   - Open index.html in VS Code
   - Right-click → "Open Preview" (Ctrl+Shift+V)

TROUBLESHOOTING:
----------------
1. If hamburger menu doesn't work:
   - Check browser console for errors (F12 → Console)
   - Ensure all files are in same folder

2. If map doesn't show:
   - Check internet connection
   - Map loads from external CDN

3. If website looks wrong:
   - Clear browser cache (Ctrl+Shift+Delete)
   - Try different browser
